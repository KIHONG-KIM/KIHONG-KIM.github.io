기능의학적 접근으로 자폐증 회복 사례가 보고되고 있다. 비타민B군, 장내 유산균, 케톤식이, 영양소 보충을 통해 1~2년 내 치유에 가까운 회복이 관찰되었다. 이를 생화학적으로 이해하고 시각화하는 교육 도구 제작을 위한 학습 정리.

---

## 핵심 가설

소장에서의 **중금속 해독 불량** → 대사 퇴행 → 신경 기능 저하

원인은 다양하다:
- 수술 후 퇴행
- 환경적 중금속 노출
- 유전적 효소 변이 (MTHFR 등)

이 공통점은 **대사경로 기능 저하**이며, 영양소 보충으로 경로를 복구하면 회복 가능하다는 것이 기능의학의 주장이다.

---

## 인체 대사경로 규모

KEGG 데이터베이스 기준 **500개 이상**의 대사경로가 존재한다. 핵심은 다음 카테고리로 분류된다.

---

## 에너지 대사 (Energy Metabolism)

### 해당과정 (Glycolysis)
```
포도당 → 피루브산 → 아세틸-CoA
필요 영양소: B1(티아민) - 피루브산 탈수소효소의 보조인자
```

### TCA 회로 (크렙스 회로)
```
아세틸-CoA → [8단계 반응] → ATP + NADH + FADH2
필요 영양소: B1, B2, B3, B5, Mg²⁺
```

### 케톤체 대사
```
지방산 → 아세토아세테이트 → β-하이드록시부티레이트
→ 뇌의 대체 에너지원 (포도당 불필요)
```
케톤식이가 자폐 증상에 효과적인 이유: 미토콘드리아를 우회하는 효율적 에너지 공급

### 베타산화 (β-Oxidation)
```
지방산 → 아세틸-CoA (미토콘드리아)
필요 영양소: B2, B3, 카르니틴
```

---

## 메틸화 & 해독 대사 (Methylation & Detox)

### 메틸화 사이클 (Methylation Cycle) ← 자폐와 가장 직접 연관

```
메티오닌 → SAM (메틸 공여체)
    ↓
SAH → 호모시스테인(Hcy)
    ↓              ↓
  재활용          트랜스설퍼레이션 경로
(B12, 엽산 필요)
```

- **SAM**: DNA 메틸화, 신경전달물질 합성에 필수
- **MTHFR 변이**: 엽산을 활성형(5-MTHF)으로 전환 못함 → 메틸화 전체 막힘
- **B9(엽산)**: 메틸화 사이클의 핵심 재료
- **B12**: Hcy → 메티오닌 재활용에 필수

### 트랜스설퍼레이션 경로 (Transsulfuration Pathway)

```
호모시스테인 → 시스타티오닌 → 시스테인 → 글루타치온(GSH)
                                           (마스터 항산화제)
필요 영양소: B6(피리독신)
```

**글루타치온(GSH)**: 중금속과 직접 결합하여 체외 배출을 담당하는 핵심 분자

## 아미노산 & 신경전달물질 대사

```
트립토판 → 5-HTP → 세로토닌 → 멜라토닌
           (B6 필요)  (90%가 장에서 생성)

페닐알라닌 → 티로신 → L-DOPA → 도파민 → 노르에피네프린
                      (B6, Fe 필요)

글루탐산 ← → GABA
         (B6, Mg 필요)
```

세로토닌의 90%가 **장**에서 생성된다. 장내 환경이 무너지면 신경전달물질 전체가 영향받는 이유이다.

### 요소 회로 (Urea Cycle)
```
암모니아 → 요소 → 신장 배출
필요: 아르기닌, 아스파르트산
```
장내 dysbiosis → 암모니아 과잉 생성 → 뇌 독성

---

## 장내 미생물 대사 (Gut Microbiome)

### 단쇄지방산 (SCFA) 생산
```
식이섬유 → [장내세균] → 부티레이트 + 프로피오네이트 + 아세테이트
```
- **부티레이트**: 장 점막 세포의 주 에너지원, 장 누수 방지
- **프로피오네이트**: 과잉 시 자폐 관련 행동 유발 가능 (동물 연구)

### 비타민 합성
장내 세균이 합성하는 영양소: B12, K2, 엽산, B7(바이오틴)

## 중금속이 대사를 방해하는 기전

| 중금속 | 주요 타깃 | 영향 |
|--------|-----------|------|
| 수은(Hg) | 메틸화 효소, 글루타치온 | 메틸화 전체 억제, 항산화 고갈 |
| 납(Pb) | 헴 합성 효소 | 포르피린 축적, 미토콘드리아 장애 |
| 비소(As) | 피루브산 탈수소효소 | TCA 회로 입구 차단 |
| 알루미늄(Al) | 혈뇌장벽 | 신경독성, 신경염증 |

**핵심 악순환:**
```
중금속 축적 → 글루타치온 고갈 → 중금속 해독 불가 → 더 많은 축적
```
글루타치온 보충 / NAC 투여 / 메틸화 지원이 이 악순환을 끊는 전략이다.

## 비타민 B군 역할 요약

| 비타민 | 주요 대사 역할 | 결핍 영향 |
|--------|--------------|-----------|
| B1 (티아민) | TCA 회로, 피루브산 대사 | 에너지 생산 차단, 신경 손상 |
| B2 (리보플라빈) | FAD 전구체, 메틸화 재생 | 산화스트레스 증가 |
| B3 (나이아신) | NAD+ 전구체, 에너지 대사 | 세포 에너지 고갈 |
| B5 (판토텐산) | CoA 전구체 | 지방산·에너지 대사 장애 |
| B6 (피리독신) | 아미노산 대사, GABA 합성 | 신경과민, 경련 |
| B7 (바이오틴) | 지방산 합성, 포도당 신생합성 | 혈당 불안정 |
| **B9 (엽산/5-MTHF)** | **메틸화 사이클 핵심** | **메틸화 전체 차단** |
| **B12 (코발라민)** | **메틸화, 미엘린 합성** | **신경 퇴행, 호모시스테인↑** |

## 장-뇌 축 (Gut-Brain Axis)

```
[장내세균] → SCFA, 신경전달물질, LPS
     ↓
[장 면역] → 미주신경 → [뇌]
     ↓
[혈뇌장벽] → 염증 물질 통과
     ↓
[소교세포 활성화] → 신경염증 → 행동/인지 변화
```

**장 누수(Leaky Gut) 발생 시:**
- LPS(내독소) → 혈류 유입 → 전신 염증
- 미소화 단백질 → 면역 과잉반응
- 중금속 흡수 증가

## 기능의학적 회복 프로토콜 (대사경로 관점)

```
케톤식이 → 미토콘드리아 우회 에너지 공급 (손상된 해당과정 보완)
비타민B군 → 메틸화 사이클 + TCA 회로 복구
유산균 보충 → dysbiosis 교정 → SCFA 회복 → 장벽 복구
글루타치온/NAC → 중금속 배출 재개
마그네슘/아연 → 300개 이상의 효소 기능 복구
```
