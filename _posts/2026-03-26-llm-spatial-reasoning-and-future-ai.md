## LLM은 "뇌만 있고 눈이 없다"

LLM은 세상을 토큰 시퀀스로 봅니다. 그리드를 볼 때도 `[[0,1,2],[3,4,5]]` 처럼 숫자 나열로 인식합니다. 공간적 관계, 대칭, 패턴을 **직접 지각(perceive)하지 못하고** 추론을 통해 간접적으로 도달합니다.

인간은 그리드를 보는 순간 "아, 대칭이네"를 즉각 인식하지만, LLM은 언어로 설명받는 맹인이 듣고 추측하는 수준입니다.

---

## 시각피질과 선험적 구조

인간의 시각 처리는 한 단계가 아닙니다.

```
망막 → V1 (엣지, 선) → V2 (윤곽) → V4 (색, 형태) → IT (객체 인식)
```

각 단계가 점점 추상적인 특징을 뽑아냅니다. 그리고 이 처리 과정 아래에는 **선험적 구조(A Priori Structure)** 가 있습니다. 학습하기 전부터 갖고 태어나는 지각 원리들입니다.

**게슈탈트 법칙:**
- **근접성** — 가까운 것끼리 묶어서 봄
- **유사성** — 비슷한 것끼리 묶어서 봄
- **대칭성** — 대칭을 즉각적으로 인식

인간은 이 원리들이 자동으로 작동합니다. 배운 게 아니라 그냥 그렇게 보입니다. 하드웨어 수준입니다.

---

## 자연에서 찾은 힌트들

### 박쥐의 반향정위
박쥐는 수동적으로 듣는 게 아니라 **능동적으로 탐색**하며 공간 모델을 직접 구축합니다. 언어로 설명받는 게 아니라 신호를 쏘고 돌아오는 패턴으로 세계를 이해합니다.

### Paul Bach-y-Rita의 실험
혀에 카메라 신호를 전기 자극으로 전달했더니, 시각장애인이 결국 "보는" 것처럼 공간을 인식하게 됐습니다. 뇌는 신호의 형태가 아니라 **정보의 구조**에 적응합니다.

### 개미 군집
개미 한 마리는 멍청합니다. 하지만 군집은 최단경로를 찾고, 먹이를 분배하고, 위기에 대응합니다. 언어 없이, 중앙 제어 없이.

> 지능이 개체에 있는 게 아니라 **관계와 상호작용**에 있다는 힌트

### 예측 코딩 (Predictive Coding)
뇌는 감각 기관에서 올라오는 신호를 수동적으로 처리하는 게 아니라, 항상 "다음에 이런 게 올 것이다"를 예측하고 **오차만 수정**합니다. 현실은 수신하는 게 아니라 예측하는 것입니다.

## 바인딩 문제 (Binding Problem)

언어를 처리할 때와 공간을 처리할 때 뇌의 다른 영역이 활성화되지만, 그것이 **어떻게 통합되어 하나의 이해가 되는지**는 아직 미스터리입니다.

현재 AI 개발은 사실상:
```
블랙박스인 인간을
블랙박스인 신경망으로
흉내내는 것
```

왜 잘 되는지도, 왜 실패하는지도 완전히 설명하지 못합니다.

---

## LLM이 언어를 잘 하는 이유

> LLM이 언어를 잘 하는 건, 언어가 **이미 인간이 공간/감각을 언어로 압축해놓은 결과물**이기 때문일 수 있습니다.

LLM은 원본 경험이 아니라 **인류가 수천 년간 만들어온 압축본**을 학습한 것입니다. 공간 추론이 약한 이유도 여기 있습니다. 언어로 완전히 표현되지 않는 정보는 처음부터 학습 데이터에 없었던 것입니다.

## 진짜 Grounding의 조건

카메라만 붙이는 것과 진짜 grounding의 차이:

```
카메라만 붙이기
→ 이미지를 토큰으로 변환해서 LLM에 입력
→ 여전히 맹인이 사진 설명 듣는 것

진짜 grounding
→ 시간의 흐름 속에서 예측하고 틀리고 수정하는 과정
→ 행동의 결과를 경험
→ 세계 모델이 내부에 형성됨
```

**경험 없는 관찰은 반쪽짜리입니다.**

---

## 미래 AI의 그림

```
카메라 (공간+시간 입력)
     +
능동적 행동 (로봇, 시뮬레이션)
     +
예측 코딩 구조 (오차로 학습)
     +
언어 (추상화, 소통)
```

이 네 가지가 **같은 표현 공간에서 seamless하게** 연결될 때 처음으로 진짜 grounding이 됩니다.

## LLM 외 대안 구조들

| 구조 | 핵심 아이디어 |
|---|---|
| GNN (그래프 신경망) | 관계를 직접 표현, 노드=셀 엣지=공간관계 |
| 프로그램 합성 | 규칙을 실행 가능한 코드로 생성하고 검증 |
| 신경-기호 혼합 | 신경망의 지각 + 기호 논리의 추론 |
| RL + 능동 탐색 | 박쥐처럼 능동적으로 탐색하며 공간 모델 구축 |
| 메모리 증강 신경망 | 외부 메모리에 중간 단계 기록하며 처리 |

가장 유망한 방향은 **프로그램 합성 + 신경-기호 혼합**의 조합입니다.

## 결론

ARC(Abstraction and Reasoning Corpus) 문제가 어려운 이유는 **경험 없이 패턴만 보고 물리적 직관을 요구**하기 때문입니다. 가장 중요한 발견들이 익숙함을 특별한 시선으로 본 것처럼, AI의 다음 도약도 자연 어딘가에 이미 힌트가 있을 것입니다.
